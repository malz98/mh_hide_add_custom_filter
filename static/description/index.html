Configuration

  • Simply install the module from the Odoo app store, and the new functionality will be added.
  • Navigate to User Groups (Settings > Users & Companies > Groups).
  • Check the user group named "Hide Add Custom Filter Menu" for which you want to hide or show the add custom filter menu.
  • Enable or disable the visibility setting as per your requirements.
  • Save changes and refresh the page – the filter menu will now be displayed only for authorized users!
